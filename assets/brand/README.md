# Alfred Brand Assets

Use `alfred-logo-transparent.png` for README headers, directory submissions,
presentations, and places where the background is unknown.

Use `alfred-logo.png` when a dark square tile is useful, such as favicons,
app icons, or small previews that need a fixed background.

Use `alfred-logo-light.png` for light cards that need a fixed square canvas.

The docs-site social image is generated from
`site/scripts/generate-og.mjs` and published at
`site/public/brand/alfred-og.png`.
